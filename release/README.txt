Snake - SFML

Windows build package for itch.io.

Contents:
- SnakeSFML.exe
- Required SFML runtime DLLs
- Required MinGW runtime DLLs

Controls:
- Arrow keys or WASD to move
- Enter to start or restart
- Esc to quit

Start screen options:
- A / D: change background theme
- Q / E: change snake color
- W / S: change difficulty

If the game does not start, make sure all DLL files remain in the same folder as SnakeSFML.exe.
